## Overview

**m2BIM** is a structural BIM practice dedicated to advancing the field through **high-quality information management**, **automation**, and adherence to **international BIM standards**.

This GitHub organization hosts **m2LAB** ‚Äî an open research space for developing and sharing practical BIM tools and workflows that are actively used in real-world projects.
